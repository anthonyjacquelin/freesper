Icon Assets Needed
==================

Please create/add the following icon files:

1. iconTemplate.png (16x16 or 32x32) - For menu bar tray icon
   - Should be a template image (black with alpha channel)
   - macOS will auto-adjust for light/dark mode

2. icon.icns - For app icon
   - Create from a 1024x1024 PNG using:
     - iconutil (built into macOS)
     - Or online converters

Quick way to create iconTemplate.png:
- Use any simple microphone or speech bubble icon
- Make it monochrome (black on transparent)
- Save as PNG at 32x32 or 16x16

For now, the app will work without icons but may show a blank space in the menu bar.
