# Sons Audio de Freesper

Ce dossier contient les fichiers audio utilisés pour le feedback sonore lors de l'enregistrement.

## Fichiers

### Sons de démarrage
- `Start1.m4a` - Son de démarrage (variation 1)
- `Start2.m4a` - Son de démarrage (variation 2)
- `Start3.m4a` - Son de démarrage (variation 3)
- `Start4.m4a` - Son de démarrage (variation 4)

### Sons d'arrêt
- `Stop1.m4a` - Son d'arrêt (variation 1)
- `Stop2.m4a` - Son d'arrêt (variation 2)
- `Stop3.m4a` - Son d'arrêt (variation 3)
- `Stop4.m4a` - Son d'arrêt (variation 4)

## Fonctionnement

L'application sélectionne aléatoirement un son parmi les variations disponibles pour chaque événement (démarrage/arrêt), ce qui rend l'expérience plus naturelle et moins monotone.

## Source

Les fichiers audio ont été extraits de SuperWhisper, une application similaire de transcription vocale.

## Format

- Format : M4A (AAC)
- Durée : ~0.2-0.5 secondes
- Volume : Ajusté à 60% dans le code pour éviter d'être trop fort
