# Freesper Audio Sounds

This folder contains audio files used for sound feedback during recording.

## Files

### Start Sounds
- `Start1.m4a` - Start sound (variation 1)
- `Start2.m4a` - Start sound (variation 2)
- `Start3.m4a` - Start sound (variation 3)
- `Start4.m4a` - Start sound (variation 4)

### Stop Sounds
- `Stop1.m4a` - Stop sound (variation 1)
- `Stop2.m4a` - Stop sound (variation 2)
- `Stop3.m4a` - Stop sound (variation 3)
- `Stop4.m4a` - Stop sound (variation 4)

## How It Works

The app randomly selects a sound from the available variations for each event (start/stop), making the experience more natural and less monotonous.

## Format

- Format: M4A (AAC)
- Duration: ~0.2-0.5 seconds
- Volume: Set to 60% in code to avoid being too loud
